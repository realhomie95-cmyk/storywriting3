# Caption Methodology Reference

Full guide for writing Facebook captions that accompany **4:5 drama photo prompts** (Nano Banana 2).

---

## Core Goal

Write the caption as the **beginning of a dramatic story**, not a summary of the image.
The reader should feel like they're reading the first paragraph of a real emotional Facebook post.
Create curiosity, social tension, and a reason to click through to the **full story article** — then stop before the payoff.

**Workflow:** Post photo + caption on Facebook → user comments article link → reader lands on website for full story.

---

## Specs

| Property | Value |
|---|---|
| Length | 1000–1200 characters (count **characters**, not words) |
| Language | English |
| Structure | 3 paragraphs + 1 blank line + CTA |
| Dialogue | Only quote lines that appear **verbatim in the photo prompt or full story plan** — never invent for caption alone |
| Ending | Never reveal the full twist |
| CTA purpose | Drive comment engagement (`MORE` / `YES` / `NEXT`) before user posts article link in comments |

---

## The 10 Opening Styles

Choose the single best fit for each story. Rotate styles — no two consecutive captions use the same one.

### 1. Dialogue Slap Opener
Open with a strong insult or accusation line from the story.
Best for: verbal humiliation hooks.

### 2. Silent Witness Opener
Open with the room or crowd watching something unfair — before anyone speaks.
Best for: public humiliation with bystanders.

### 3. Object Clue Opener
Open with a strange or significant object — dog tag, ring, envelope, worn shoes.
Best for: object-pivot twists.

### 4. Accuser POV Opener
Open from inside the judger's confidence.
Best for: reversals where certainty makes the fall harder.

### 5. Victim Restraint Opener
Open with the judged person staying calm instead of fighting back.
Best for: patience/silence as the hook.

### 6. Room Goes Wrong Opener
Open with a sound, movement, or reaction that changes the mood.
Best for: secondary character's reaction as first clue.

### 7. Visual Reveal Opener
Open with an action that seems impossible before explaining context.
Best for: striking visual moments (dog at feet, player kneeling).

### 8. Deadline Pressure Opener
Open with countdown, auction closing, ceremony ending.
Best for: irreversible moments about to happen.

### 9. Wrong Apology Opener
Open with forced apology before power shifts.
Best for: defensive position that inverts.

### 10. Moral Choice Opener
Open with unexpected selfless choice before stakes are clear.
Best for: small kindness triggering the twist.

## Paragraph Structure

### Paragraph 1 — Emotional Hook
Establish social conflict in the first two sentences. Get to the unfairness fast.

### Paragraph 2 — Build the Pressure
Deepen humiliation or tension. Include **one specific visual detail** from the photo prompt (worn shirt, oil on hands, cracked sole, badge clipped backward).

### Paragraph 3 — Begin the Twist
Let something shift — stop before full payoff. Use action, reaction, object clue, or one exact dialogue line. End one step from knowing.

### Blank Line + CTA
One blank line before CTA. CTA must contain exactly one of: `'MORE'`, `'YES'`, `'NEXT'`

**Allowed CTA formats:**
```
Drop 'MORE' if you want to know what happens next!! 👇
Comment 'YES' if you want the full story!! 👉
Type 'NEXT' if you want the next part!! ⬇️
Drop 'MORE' if you want to know what happens next!! ⤵️
Comment 'YES' if you want the full story!! ➡️
Type 'NEXT' if you want the next part!! 🔗
```

## Caption ↔ Photo ↔ Full Story Alignment

| Element | Caption | Photo prompt | Full story |
|---------|---------|--------------|------------|
| Opening beat | Hooks with conflict | Shows peak-tension moment | Opens matching caption energy, then expands |
| Twist payoff | Hidden | Not shown | Fully revealed with satisfying ending |
| Dialogue | Max 1 quoted line if in prompt | Optional in scene description | Full dialogue allowed |
| Visual detail | One from photo prompt | Primary source | Enriched, not contradicted |

---

## The Dialogue Rule

**Allowed**: Quote a line present verbatim in the photo prompt or agreed story plan.
**Not allowed**: Invent dialogue for caption that won't appear in the full story.
**If no dialogue fits**: Write as narration.

---

## Quality Checklist

- [ ] 1000–1200 characters (count them)
- [ ] Reads like story opening, not image description
- [ ] Paragraph 2 has one visual detail from photo prompt
- [ ] Paragraph 3 begins twist without completing it
- [ ] Quotes match source verbatim
- [ ] CTA uses only MORE / YES / NEXT
- [ ] Opening style differs from previous caption in batch

---

## Anti-Patterns

- **"The room went silent"** as closing line
- **"His/her face changed"** as twist signal — use specific action
- **Summarizing the prompt** — caption is a story opening
- **Revealing twist category** (e.g., "she owned the building")
- **Same structure** as previous caption in batch
